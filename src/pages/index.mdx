    <p>
      Modern computer systems contain multiple components that need access to a common
      communication channel called the <strong>system bus</strong>. The CPU, memory
      controller, DMA controller, storage devices, and peripherals may all attempt to
      use the bus simultaneously (Stallings, 2016). Without a mechanism to control access,
      data collisions and system instability would occur (Null &amp; Lobur, 2014).
    </p>

    <p>
      This exhibit explains how <strong>bus arbitration</strong> decides which device
      gets control of the bus when several devices request access at the same time.
